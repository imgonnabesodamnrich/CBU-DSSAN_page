극심한 불균형 데이터 하에서의 가스터빈 고장 진단 프레임워크

논문 제목: Highly imbalanced fault diagnosis of gas turbines via clustering-based downsampling and deep siamese self-attention network

저널: Advanced Engineering Informatics, Vol. 54, 2022

DOI: 10.1016/j.aei.2022.101725

1. 연구 배경

가스터빈은 항공기 및 산업 분야의 핵심 장비입니다. 실제 운용에 있어서 가스터빈은 매우 높은 신뢰성을 가지고 있기 때문에, 과거 모니터링 데이터의 대다수는 정상 상태 데이터이며 고장 데이터는 극히 드뭅니다. 이러한 데이터 분포는 전형적인 "극심한 불균형 데이터(Highly Imbalanced Data)"를 구성합니다.

불균형 데이터 환경에서 전통적인 딥러닝 분류 모델은 다수 클래스(정상 샘플)에 더 많은 주의를 집중하는 경향이 있습니다. 모델이 모든 입력을 정상으로 예측하더라도 전체 정확도는 매우 높게 유지되기 때문에, 결과적으로 고장 샘플의 미진단율이 급격히 상승하게 됩니다.

또한, 미세 고장 샘플은 특징 공간에서 정상 샘플과 고도로 중첩(Class Overlapping)되는 경우가 많아 일반적인 분류 경계로는 이들을 효과적으로 구분하기 어렵습니다. 다차원 모니터링 데이터를 처리할 때 전통적인 시계열 모델은 시간적 상관관계 추출이 불충분하다는 문제에도 직면합니다. 따라서 본 논문은 데이터층과 모델층의 최적화를 결합한 진단 프레임워크인 CBU-DSSAN을 제안합니다.

이 방법은 데이터 균형을 위한 클러스터링 기반 다운샘플링(CBU)과 특징 추출 및 분류를 위한 심층 샴 자가 주의 네트워크(DSSAN)라는 두 가지 주요 기술로 구성됩니다.

2.1 데이터층: 클러스터링 기반 다운샘플링 (CBU)

데이터 불균형에 대응하는 가장 직관적인 방법은 다수 클래스를 다운샘플링(정상 샘플 수 감소)하는 것입니다. 그러나 전통적인 무작위 다운샘플링은 핵심적인 대표 정보를 잃기 쉽습니다. 논문은 K-means 클러스터링 기반의 다운샘플링 전략을 채택했습니다:

  1. 방대한 양의 정상 샘플에 대해서만 클러스터 분석을 수행하여 여러 클러스터(Cluster)로 나눕니다.
  2. 각 클러스터의 중심점(Cluster Center)을 정상 샘플의 대표로 추출합니다.
  3. 이러한 대표 정상 샘플과 희소한 고장 샘플을 결합하여 새로운 클래스 균형 훈련 세트를 구성합니다.

이 방식은 정상 샘플의 절대적인 수를 줄이면서도 정상 상태의 핵심 데이터 분포 특성을 보존합니다.

2.2 모델층: 심층 샴 자가 주의 네트워크 (DSSAN)

균형 잡힌 데이터셋을 얻은 후, 모델은 샘플 중첩과 특징 추출 문제를 해결해야 합니다. DSSAN 모듈은 두 가지 핵심 메커니즘을 포함합니다:

1. 샴 네트워크 아키텍처 (Siamese Network)
단일 샘플을 직접 분류하는 전통적인 모델과 달리, 샴 네트워크는 "샘플 쌍"을 입력으로 받습니다. 훈련 시 모델의 목표는 이 쌍이 동일한 범주에 속하는지 여부를 판단하는 것입니다. 대비 손실 함수(Contrastive Loss)를 도입함으로써, 모델은 훈련 과정에서 동일 클래스의 샘플을 특징 공간에서 가깝게 당기고, 서로 다른 클래스의 샘플을 멀리 밀어냅니다. 이 메커니즘은 특징 공간에서 정상 샘플과 고장 샘플 사이의 거리를 효과적으로 넓혀 클래스 중첩으로 인한 오판 문제를 완화합니다.

2. 자가 주의 메커니즘 (Self-Attention Mechanism)
가스터빈 모니터링 데이터는 전형적인 다차원 시계열 데이터입니다. 샴 네트워크의 특징 추출 모듈로서 자가 주의 네트워크(SANet)는 서로 다른 위치와 차원의 데이터 포인트에 대해 동시에 전역적인 주의를 기울일 수 있습니다. 다중 헤드 자가 주의 메커니즘(Multi-head Self-Attention)을 통해 모델은 시간 차원에서의 데이터 의존성과 진화 법칙을 보다 포괄적으로 포착할 수 있습니다.

3. 실험 설계 및 데이터 처리

이 연구는 아시아 소재 모 항공사의 실제 운항 데이터를 사용했습니다. 직접 측정된 파라미터는 환경과 추력의 간섭을 받으므로, 실험은 OEM(주문자 상표 부착 생산자)에 의해 변환된 기준 편차값을 특징 파라미터로 선택했습니다. 주요 항목은 다음과 같습니다:

  • 배기 가스 온도 마진 (EGTM)
  • 배기 가스 온도 편차 (DEGT)
  • 코어 엔진 회전 속도 편차 (DN2)
  • 연료 유량 편차 (DFF)

시계열 특성을 고려하여 슬라이딩 윈도우 방식을 사용하여 샘플을 구성했으며, 단일 실험의 무작위성을 제거하기 위해 5-겹 교차 검증을 사용했습니다. 데이터 세트에는 984개의 정상 샘플과 소량의 배기 온도 지시 고장(EIF) 및 총온도 센서 지시 고장(TSIF) 샘플이 포함되었습니다.

4. 결론

주요 딥러닝 네트워크(ConvNet, RNN, LSTM, GRU 등) 및 일반적인 리샘플링 방법(SMOTE 오버샘플링, 일반 클러스터링 다운샘플링)과 비교하여 실험은 객관적인 성능 데이터를 도출했습니다:

  1. 정확도 향상: 균형 정확도(Balanced Accuracy, 각 범주의 인식률을 종합 고려) 지표에서 CBU-DSSAN은 92.26%를 기록하여, 단독 베이스라인 모델 및 다른 샘플링 전략을 결합한 대조군보다 명확하게 우수했습니다.
  2. 특징 분리 효과: 2차원 특징 시각화(t-SNE) 결과, 전통적인 모델이 추출한 특징 공간에서는 고장 샘플과 정상 샘플이 심하게 섞여 있었습니다. 반면 CBU-DSSAN이 추출한 공간에서는 서로 다른 범주의 샘플들이 명확한 경계를 보여주어 샴 네트워크 대비 손실의 유효성을 입증했습니다.
  3. 추론 효율성: 샴 네트워크의 구조는 비교적 복잡하지만, 훈련 후 단일 테스트 샘플에 대한 추론 시간은 약 1.33밀리초로 산업 현장의 실시간 모니터링 요구사항을 충분히 충족합니다.

이 방법은 클래스 중첩이 존재하는 극심한 불균형 산업 고장 데이터를 처리할 때, 데이터 수준의 데이터 압축(대표성 유지)과 알고리즘 수준의 거리 메트릭 학습(클래스 간 간격 확대)을 결합하는 것이 매우 효과적인 경로임을 시사합니다.
